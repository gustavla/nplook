npzlook
=======

Command-line tool that peeks into a numpy `npy` or `npz` file and summarizes its contents.

quick install
-------------

    [sudo] pip install nplook

install from github
-------------------

    git clone git@github.com:gustavla/nplook.git
    cd nplook
    [sudo] python setup.py install

usage
-----

    $ nplook data.npy 
    $ nplook *.npz
